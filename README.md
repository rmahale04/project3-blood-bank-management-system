# 🩸 Blood Bank Management System

A full-stack web application developed to streamline blood donation and blood request management through a centralized platform. The system enables efficient management of donors, hospitals, blood inventory, donations, and requests while reducing manual record keeping.

---

## 📖 Project Overview

The Blood Bank Management System was developed as an academic group project to demonstrate practical knowledge of **Web Development, Database Design, Software Engineering, and Relational Database Management**.

The application digitizes the blood donation workflow by maintaining donor information, managing blood inventory, processing hospital requests, and providing administrative controls through a role-based system.

---

## ❓ Problem Statement

Traditional blood bank management often relies on manual record keeping, making it difficult to:

- Maintain donor records
- Track available blood stock
- Process blood requests efficiently
- Monitor donation history
- Maintain accurate inventory

This project provides a centralized web-based solution to automate these processes.

---

## 🎯 Objectives

- Digitize blood bank operations
- Maintain accurate donor records
- Manage blood inventory efficiently
- Track blood donations
- Process hospital blood requests
- Reduce manual paperwork
- Improve data consistency
